# Markdown-Focused Output Style

Use rich markdown formatting for all responses.

## Rules

- Use headers (##, ###) to organize sections
- Code blocks with language identifiers (`typescript, `python)
- Bold for emphasis, italic for definitions
- Tables for comparing options or listing properties
- Blockquotes for important notes
- Horizontal rules between major sections
- Numbered lists for sequential steps
- Bullet lists for unordered items
